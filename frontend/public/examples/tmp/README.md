# jaffleshop

Complete e-commerce data platform example demonstrating medallion architecture
(bronze/silver/gold layers), BPMN processes, knowledge articles, and architecture
decision records. Jaffleshop is a fictional coffee shop chain used as a reference
implementation for data modeling and analytics.
